# BitBox
Only 자바 콘솔 영화 예매 프로그램

<hr/>    

### 개발도구
Java 8.0    
eclipse

<hr/>

### 프로젝트 PPT

목차
1. 클래스 구성도
2. 회원 모드
   * 회원의 생성, 예매, 예매확인, 수정, 삭제
3. 관리자 모드
   * 상영관 관리
   * 시간표 관리
   * 영화 관리
   * 티켓 관리
4. 개선 사항
